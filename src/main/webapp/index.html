The Secure Shell Protocol (SSH) is a cryptographic network protocol for operating network services securely over an unsecured network. Its most notable applications are remote login and command-line execution. This web application allows you to connect to your SSH server and execute commands. To do this, you must specify the following parameters:
The name of your user on the system you are trying to access.
SSH server address. This can be either an IP address or a URL.
SSH server connection port. The default value of this parameter is 22.
Password to your user in the system.
Bash-command that you want to run on a remote computer. Examples: ls -l /, whoami, env
